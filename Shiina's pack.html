A bit about Shiina's pack

This texture pack was the first spark to an immense forest fire that is this hobby, I was unsatisfied with the color choice of the mincraft diamond armor, and so I wanted to change it. Fortunatly in doing so i had a great time and it became one of my hobbys, I also had a bit of fun retexturing some other items in the texture pack.

Also if you're wondering where the name "Shiina" came from, well it's my favorite name so i decided to name my first pack after it.
