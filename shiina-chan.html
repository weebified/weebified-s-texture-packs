Commands


Prefix is "Shiina please"

Moderation


  • Ban:

    Bans the user metionned, syntax is "Shiina please ban [insert user]"

  • Kick:

    Kicks the user mentionned, syntax is "Shiina please kick [insert user]

  • Mute:

    Mutes the user mentionned, syntax is "Shiina please mute [insert user] [amout of time, optional]

  • Unmute:

    Unmutes the user mentionned, syntax is "Shiina please unmute [insert user]

Others


  • Cookie:

    Gives a cookie to the mentionned player, syntax is "Shiina please cookie [insert user]"

  • Keyboard:

    Sends a picture from the subreddit r/MechanicalKeyboards, syntax is "Shiina please keyboard"

  • Shave:

    Shaves the player mentionned, syntax is "Shiina please shave [insert user]"

  • Test:

    It's legit just to know if the bot is online, syntax is "Shiina please test"

  • Help:

    Sends a link to this page, syntax is "Shiina please help"
