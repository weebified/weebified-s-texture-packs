If you're curious as to what changed, well I basically made it alot simpler. Instead of looking for a prefix and taking the rest of the message as a command, Kuro takes the message as a command and checks if that command exists. I also took the opportunity to make some general improvements.

Universal cooldown: 5seconds

    Test to see if the bot is online, syntax is "kuro please test", "kuro please c'mere", "kuro please cmere" or "kuro please come here"

    Sends an embed with a link to my website, litterally an advertisement command, syntax is "advert"

    Greets the person mentionned in the command, syntax is "hello [person]"

  • Neko picture command

    Sends a picture of a catgirl, syntax is "neko pic", "neko pciture", "neko photography" or "neko photo"

    Just gonna put a disclaimer, please do not execute this command if you can't handle bright flashing colors. That said, the syntax is "stroke"
