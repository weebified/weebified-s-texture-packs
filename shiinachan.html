Shiina-chan
weebified's Discord bot

Prefix: Shiina please

Moderation commands

  • Ban command

    Bans the user mentionned, syntax is "Shiina please ban @user"

  • Kick command

    Kicks the user mentionned, syntax is "Shiina please kick @user"

  • Mute command

    Mutes the user mentionned, syntax is "Shiina please mute @user [time, optional]"

  • Unmute command

    Unmutes the user mentionned, syntax is "Shiina please unmute @user"

Other commands

  • Celebrate command

    Congratulates the user mentionned, syntax is "Shiina please celebrate @user"

    cooldown: 10 seconds

  • Cookie command

    Gives a cookie to the user mentionned, syntax is "Shiina please cookie @user"

    cooldown: 10 seconds

  • Fact command

    Gives out a random fact, syntax is "Shiina please fact"

    cooldown: 10 seconds

  • Help command

    Sends a link to this page, syntax is "Shiina please help"

    cooldown: 10 seconds

  • Keyboard command
